Privacy Policy

Last updated:

1. Overview

This Privacy Policy explains how Trizly Operations collects, stores, and processes information for Lumi and all associated services.

2. Data We Collect

  • Discord IDs (users, roles, channels, guilds)
  • Bot configuration data
  • Moderation logs (warnings, bans, executor, target, timestamps)
  • Error logs, crash reports, and performance metrics
  • Optional API integration details (e.g., ER:LC keys)

3. How Data Is Used

  • Operate Lumi’s features and commands
  • Persist server configuration
  • Improve stability, detect abuse, and provide support
  • Troubleshoot errors or misconfigurations

4. Legal Basis

Data is processed based on:

  • Legitimate interests—service functionality, safety, prevention of abuse
  • User consent—when enabling optional integrations or settings

5. Data Sharing

We do NOT sell your data.

We only share data with:

  • Discord (normal bot operation)
  • Roblox/ER:LC (only if integrations are enabled)
  • Cloud hosting partners

6. Retention

Logs and data are retained for a reasonable duration for support, then automatically removed where applicable.

7. Security

We implement modern security practices but cannot guarantee absolute safety of transmitted data.

8. Your Rights

  • Request removal of your server’s data
  • Opt out by removing the bot
  • Request configuration resets

9. Children

We do not knowingly store data from users younger than Discord’s minimum age requirement.

10. Changes

This policy may be updated. Continued use of Trizly services means you accept the new version.

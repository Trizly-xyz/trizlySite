1. Overview

This Privacy Policy explains how Trizly Operations (“we”, “us”) collects, uses, stores, and protects data processed through Lumi and all Trizly services.

2. Data We Collect

  • Discord IDs (users, roles, channels, guilds)
  • Server configuration settings
  • Bot logs (actions performed, timestamps, executor, target)
  • Optional integrations (e.g., ER:LC API keys if provided)
  • Web dashboard session identifiers
  • Error logs and analytics

3. How We Use Data

  • Operate Lumi and Trizly services
  • Provide moderation and automation features
  • Improve stability and detect abuse
  • Enable server configuration persistence
  • Troubleshoot errors

4. Legal Basis

We process data under legitimate interests (functionality & security) and user consent (settings you configure).

5. Data Sharing

We do not sell your data.

We only share data with:

  • Discord (normal bot operations)
  • Roblox/ER:LC API (if you enable integrations)
  • Cloud hosting providers (for processing the bot)

6. Data Retention

Bot logs are retained for a reasonable period for support and are automatically pruned.

7. Security

We use industry-standard methods to protect data, but cannot guarantee absolute security.

8. Your Rights

  • Request server data deletion
  • Opt out by removing the bot
  • Request configuration wipes

9. Children

We do not knowingly collect data from children under the minimum Discord age limit.

10. Changes

We may update this Privacy Policy. Continued use means acceptance of updates.

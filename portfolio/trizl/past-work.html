Past Work

A selection of projects and systems I’ve built over the last few years. Some began as academic work, others as personal experiments.

Volcanic Mapping Platform • volcano-alerts.pages.dev

Web Platform • Live Data • Mapping

Originally built for a school project, this project uses Leaflet.js to visualize real-time volcanic activity data on an interactive map with data fetched by the USGS HANS API. This project is being continued as a non-volcanic interactive map showing natural disaster data.

Lumi

Roblox Verification • Custom API • Global

Lumi is the next generation Roblox account verification system, designed to provide secure and reliable verification services for Roblox communities. Built with scalability and performance in mind, Lumi leverages a custom REST API and a robust backend infrastructure to handle high volumes of verification requests globally.

More projects and details are continuously being added.

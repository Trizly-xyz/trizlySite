Past Work

A selection of projects and systems I’ve built over the last few years. Some began as academic work, others as personal experiments — all of them continue to evolve.

Volcanic Mapping Platform

Web Platform • Live Data • Mapping

Originally built for a school project, this platform has grown into an actively maintained system. It currently visualizes real-time volcanic activity across the United States using data from the USGS HANS API. The frontend is powered by Leaflet.js for interactive map rendering.

I am currently expanding the system beyond volcanoes to support a full natural disaster mapping platform capable of tracking multiple event types in real time. You can view the live version at volcano-alerts.pages.dev .

Custom APIs

Backend • Automation • Verification

A collection of backend services designed to support Discord bots, dashboards, and verification systems. These APIs handle authentication flows, data validation, logging, and automation tasks across multiple projects.

Discord Bot Systems

Automation • Moderation • Utilities
